# Tic Tac Toe Game
freecodecamp advanced front-end development projects. This is now a legacy project.

## Tech Stack
- HTML
- CSS
- JavaScript
- jQuery

## Instructions
1. Clone or download zip file. 
2. Open index.html file in your browser.
3. Click the hash tag button to put an X. The computer will take a turn and put an O in a different spot. Keep picking hash tag buttons until you get an an alert letting to you who won the game.
